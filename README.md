# 📚 DevOps Cheat Sheets

A structured collection of **daily-use DevOps commands and scripting references** that I use in real workflows.

This repository acts as a **quick reference guide** for Git, GitHub CLI, and Shell Scripting — designed to save time during development, debugging, and automation tasks.

---

## 📌 About This Repository

This repo is my personal **knowledge base of frequently used commands**, where I:

* Document commonly used Git & Linux commands
* Maintain quick references for daily DevOps tasks
* Organize learnings into structured cheat sheets
* Continuously update based on real-world usage

---

## 📂 Contents

### 🔧 Git Commands Cheat Sheet

📄 `git-commands.md`

A comprehensive list of Git and GitHub CLI commands including:

* Repository setup & configuration
* Branching & merging strategies
* Stashing, rebasing, cherry-picking
* Undo operations (`reset`, `revert`, `reflog`)
* Remote operations (`push`, `pull`, `fetch`)
* GitHub CLI (`gh`) for automation

👉 Includes practical examples and explanations for each command
📎 Reference: [git-commands.md](./git-commands.md)

### 🐚 Shell Scripting Cheat Sheet

📄 `shell_scripting_cheatsheet.md`

A structured guide covering:

* Bash basics (variables, loops, conditions)
* File operations & arguments
* Functions and scripting patterns
* Text processing tools:

  * `grep`, `awk`, `sed`, `cut`
* Useful one-liners for DevOps
* Debugging & error handling (`set -euo pipefail`)

👉 Designed for both beginners and real-world scripting usage
📎 Reference: [shell_scripting_cheatsheet.md](./shell_scripting_cheatsheet.md)
